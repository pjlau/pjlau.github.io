Adult Income Dataset

This dataset was extracted by Barry Becker from the 1994 Census database.
It is widely used for predicting whether income exceeds $50K/yr based on census data.
The original dataset can be found in UCI Machine Learning Repository
https://archive.ics.uci.edu/dataset/2/adult

Income: Annual Income

Rich: Income > 50k, Poor: Income <=50k

Poverty Rate = Poor Proportion of a Specific Demographic

For example, Male Poverty Rate = Sampled Poor Male Population / Sampled Male Population
